## 属性

| 属性名    | 描述            | 类型               | 默认值   |
| :-------- | :-------------- | :----------------- | :------- |
| wrap      | Wrap组件        | `ComponentType<P>` | `(必选)` |
| wrapProps | Wrap组件的props | `unknown`          | `-`      |
| need      | 是否需要wrap    | `boolean`          | `(必选)` |

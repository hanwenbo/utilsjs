### 属性

| 参数         | 说明   | 类型       | 默认值 |
|------------| ------ |----------|-----|
| percent    | 百分比 | `number` | `0` |
| trackWidth | 线条宽度 | `number` | `3` |
| trackColor | 线条的颜色 | `string`      | `#e5e5e5` |
| fillColor      | 填充的颜色 | `string`      | `#e5e5e5` |
